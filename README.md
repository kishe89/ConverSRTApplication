# ConverSRTApplication

이 레파지토리는 유튜브 자막 생성시 SRT 로 뽑은 자막파일을 구글 번역기로 번역시 오번역 되는 일부 문자열의 수정을 도와주는 코드 저장소입니다.



# Goal

유튜브 자막 파일을 구글 번역기로 번역시 오번역 하는 부분에 대한 올바른 포맷으로의 수정을 목표로 한다.
